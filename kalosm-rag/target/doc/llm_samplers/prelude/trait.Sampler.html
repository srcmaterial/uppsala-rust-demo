fn sample<'a>( &mut self, res: &mut dyn HasSamplerResources, logits: &'a mut Logits, ) -> Result<&'a mut Logits>

Provided Methods§

Source

fn sampled_token_id(&self) -> Option<TID>

Returns the last sampled token id if available.

A default implemenation is provided which simply returns None.

Source

fn sample_token( &mut self, res: &mut dyn HasSamplerResources, logits: &mut Logits, ) -> Result<Option<TID>>

Run the sampler and return the last sampled token id if available.

A default implementation is provided which just calls Sampler::sample followed by Sampler::sampled_token_id().
