§Locally typical sampling

An approach to sampling that attempts to maximize natural and human-like output.

See: https://arxiv.org/abs/2202.00666

Properties:

  • Modifies logits
  • Filters logits

Parameters:

  • min_keep: Minimum number of entries to keep. Setting this to 0 is not recommended. (default: 1)
  • p: Referred to as τ in the paper. It suggests using 0.2 as a value for story generation and 0.95 for “abstractive summarization” (presumably this means more factual output). (default: 1.0)
