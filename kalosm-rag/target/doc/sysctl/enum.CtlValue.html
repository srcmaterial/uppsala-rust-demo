Expand description

An Enum that holds all values returned by sysctl calls. Extract inner value with accessors like as_int().

§Example

if let Ok(ctl) = sysctl::Ctl::new("kern.osrevision") {
    if let Ok(r) = ctl.value() {
        if let Some(val) = r.as_int() {
            println!("Value: {}", val);
        }
    }
}
